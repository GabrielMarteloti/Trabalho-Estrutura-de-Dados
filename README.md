# Trabalho-Estrutura-de-Dados
Trabalho de Recursividade

Trabalho de Recursividade - Estrutura de Dados

Este repositório contém a implementação de algoritmos recursivos solicitados em aula. Implementações:

Fibonacci (Clique aqui para ver o código): Versão recursiva ingênua (demonstra a ineficiência $O(2^n)$).
Versão com, Memoização e Alocação Dinâmica, (otimiza para $O(n)$).
Caso Base: if (n <= 1) return n;
Torres de Hanói (Clique aqui para ver o código) Solução recursiva para movimentação de discos entre torres (A, B e C).
Caso Base: if (n == 1) (movimentação do último disco).
Como compilar e rodar: Utilize um compilador C (como GCC):

gcc fibonacci.c -o fibonacci
gcc hanoi.c -o hanoi
